# suBagBackUp 背包备份
## 前言
* 本插件用于自动备份玩家物品或手动备份
* 支持跨服,可搭配跨服数据同步插件使用
* ⭐支持mod和插件物品⭐
## 指令+权限
* /bbu all——备份全部在线玩家背包
* /bbu id——备份指定玩家背包
* /bbu reload——重载配置文件
* /bbu id 时间——恢复指定时间的玩家备份
* (可用时间会自动显示用tab补全)
* 权限:subagbuckup.op
